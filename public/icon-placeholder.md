# PWA Icons

Placeholder icons are currently used. For production, replace these with proper icons:

- `/public/icon-192.png` - 192x192px icon
- `/public/icon-512.png` - 512x512px icon

You can generate icons using:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- Or design tools like Figma, Sketch, etc.

The icons should:
- Be square (1:1 aspect ratio)
- Have transparent background (PNG)
- Represent the SportTrack brand
- Be recognizable at small sizes
